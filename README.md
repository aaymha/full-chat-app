# Full-Stack Chat App (Python)

Samodzielnie zaprojektowana i zbudowana aplikacja czatu w czasie rzeczywistym, oparta o **FastAPI** i **WebSockety**, z prostym frontendem w HTML, CSS i JavaScript.  


# Funkcjonalności
- Wysyłanie i odbieranie wiadomości w czasie rzeczywistym  
- Backend oparty na **FastAPI** z obsługą WebSocketów  
- Minimalny frontend w **HTML/CSS/JavaScript**  
- Prosta baza danych **SQLite** do przechowywania danych  
- Łatwy do rozbudowy i wdrożenia  

## Technologie
- **Backend**: Python, FastAPI, WebSockets  
- **Frontend**: HTML, CSS, JavaScript  
- **Baza danych**: SQLite
